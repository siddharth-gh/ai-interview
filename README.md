# AI Interview

A simple MERN-style app where a user:

1. Enters their name
2. Uploads a CV
3. Gets the CV evaluated and keywords extracted
4. Answers a 10-question text interview
5. Receives a simple performance summary

## Tech

- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Storage: In-memory for now

## Run

Install dependencies at the repo root, then run:

```bash
npm run dev
```

The app expects the client to run on port `5173` and the server on port `4000`.
